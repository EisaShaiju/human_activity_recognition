# HAR Dimensionality Reduction

A Python machine learning pipeline that reduces the dimensionality of the Human Activity Recognition (HAR) dataset using K-Means clustering and classifies physical activities using a Gaussian Naive Bayes model.

## Prerequisites

Ensure you have Python installed along with the following libraries:
* `pandas`
* `scikit-learn`
* `numpy`
* `requests`
* `beautifulsoup4`

## Installation & Usage

1. Save the script to your local machine (e.g., as `main.py`).
2. Install the required dependencies via pip:
   ```bash
   pip install pandas scikit-learn numpy requests beautifulsoup4
